How to read the barbell visualization:

1. The gray bar in the center represents the barbell.

2. Colored rectangles on each side represent weight plates.

3. The size of each rectangle is proportional to the plate's weight.

4. The legend on the top-left shows the color coding for different plate weights.

5. The visualization shows the plates needed for one side of the barbell after the drop set calculation.
